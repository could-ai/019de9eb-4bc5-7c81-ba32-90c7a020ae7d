# Springfield Academy Information App

A comprehensive Flutter application designed to provide parents, students, and prospective families with all essential information about Springfield Academy. The app serves as a digital handbook, offering quick access to everything from basic contact details to detailed curriculum information.

## Features

* **Home Dashboard**: Quick overview of the school, contact numbers, office hours, and shortcut links.
* **Admissions**: Detailed guide on requirements, age limits, required documents, and the application process step-by-step.
* **Academics**: 
  * **Curriculum**: Information on subjects by grade level (Elementary, Middle, High School) and special programs.
  * **School Hours**: Daily timetables and the complete school calendar with important dates and holidays.
* **Student Life**: 
  * **Facilities**: Overview of the library, labs, sports complex, canteen, and medical room.
  * **Uniforms**: Guidelines for daily and PE uniforms, including where to purchase them.
  * **Transport**: Bus routes, schedules, and instructions on how to register for transport services.
* **Fees & Payments**: Detailed breakdown of tuition, mandatory fees, optional services, and accepted payment methods.
* **About & Staff**: Directory of key leadership, administration, and support offices.
* **Documents & FAQ**: Centralized hub for downloadable forms, policies, and frequently asked questions.

## Tech Stack

* **Framework**: Flutter
* **Language**: Dart
* **Architecture**: Standard Flutter project structure with modular screens and a centralized `AppShell` with bottom navigation.
* **UI**: Material Design 3 with custom theming.

## Setup & Run Instructions

1. Ensure you have the Flutter SDK installed on your machine.
2. Clone this repository.
3. Run `flutter pub get` in the root directory to fetch dependencies.
4. Run the app on your preferred emulator or connected device:
   ```bash
   flutter run
   ```
